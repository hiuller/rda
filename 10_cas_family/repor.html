Descoberta no-supervisionada de famlias de consumo de alumnio no CAS-OB

Introduo

A anlise da variao do consumo de alumnio nas estaes de refino realizada para descobrir se houve alguma alteraao no processo em comparao com um perodo de referncia chamado perodo de oramento. O perodo de oramento usado como referncia para o ano de 2015 vai do dia 01/set/13 a 31/ago/14.

O objetivo deste trabalho descobrir de forma no-supervisionada quais seriam as famlias que possuem consumo de alumnio parecido de forma a serem indistintas. Esses agrupamentos (clusteres) sero usados para formalizar uma metodologia de determinao de consumo padro. O consumo padro do perodo em relao ao oramento serve para isolar o efeito do mix de produo das demais variaes permitindo descobrir se houve ou no alguma ocorrncia que precise ser corrigida. Essa informao serve para prevenir que se inicie uma investigao de um problema que no existe.

A seguir, foi criado um dendograma baseado na matriz de distncia (mdm) criada com base na transformao do p-valor do teste.t no formato:

\[ p'=\frac{1}{1+p} \]

sample <- subset(unico, select=c('a', 'b', 'pvs'))
library(reshape2)
#vou normalizar os p-valores para que o dendograma fique mais visivel
  newP <- sample$pvs
  newP <- ifelse(newP < 1e-4, 0.1, newP)
  normP <- (newP - min(newP)/(max(newP)-min(newP)))
  sample$normP <- 1/(normP+1)
  sample$pvs <- NULL

  sample.melt <- melt(sample)
## Using a, b as id variables
  mdm <- acast(sample.melt, a~b, fun.aggregate=sum, drop=FALSE)

  rdm <- as.dist(mdm)

O resultado da ANOVA atesta que os agrupamentos no so os mesmos mas isso nao garante que sejam iguais dois a dois. Assim, foi feito novo teste-t, desta vez entre os cluesteres e o resultado pode ser observado pela sada grfica. Para isto, foi criada a funo margin (abaixo) para retornar o tamaho do intervalo de confiana usando a distribuiao t de Student.

  # o p-value foi zero. Assim, vamos fazer um interval plot, igual aquela
  
  # essa funao retorna o E para construao de intervalo de confiana usando a distribuiao t de student  autor: eu em 25/05/2015 s 13h45
  margin <-function(x, conf=0.95)
  {
    n <- length(x)
    t_half_alpha <- qt(p=(conf+(1-conf)/2), df=n-1)
    return(t_half_alpha*sd(x)/sqrt(n))
  }

A sada grfica do teste-t entre as famlias apresentado pelo interval plot abaixo:

Concluses

Foi possvel obter um agrupamento de tamanho 6 para famlias de aos com base no grau de desoxidao, tipo de tratamento e enxofre mximo que tem consumo de alumnio indistinto dentro do grupo e diferente entre os grupos. Esses agrupamentos sero usados para criar um padro de consumo consistente na anlise de consumo de alumnio num ms.
